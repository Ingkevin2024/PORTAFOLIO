Mis Proyectos

ECOMOVI

MAS LIMPIO MAS VERDE

Aplicación web que permite a los usuarios se registre y gestionar el sistema de restricción vehicular voluntario. Desarrollada con HTML, CSS, php y JavaScript, ofrece una interfaz intuitiva para que los usuarios se registren, utilicen la pagina web para ganar unos puntos con su pico y placa, y contribuir a la movilidad sostenible.

PUPPY SHOP

Tienda en línea especializada en productos y accesorios para mascotas. Desarrollada con HTML, CSS, JavaScript y PHP, ofrece una experiencia con catálogo de productos intuitiva, calendario interactivo y sistema de pagos seguros. Incluye secciones de alimentos premium, juguetes, accesorios y artículos de cuidado e higiene para perros. La plataforma también cuenta con un blog informativo sobre cuidados y consejos para mascotas.
